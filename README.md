# TalTech HPC Docs Knowledge Base

This skill captures the full content of [docs.hpc.taltech.ee](https://docs.hpc.taltech.ee/) — the TalTech High Performance Computing Centre user guides.

## Usage

Load the skill in any Hermes session to get answers about TalTech's HPC infrastructure. Every answer automatically cites the source documentation page.

- Hardware Specifications (green, ada, amp, amp2, mem1tb, viz, cloud, LUMI)
- SLURM Partitions & Node Features
- Running Jobs (srun, sbatch, accounts, examples)
- Module System (Lmod / SPACK)
- Storage Paths & Quotas
- Billing Tables
- Software Catalog
- Containers (Singularity)
- MPI
- OnDemand Web Interface
- LUMI Access Procedure
- Citation & Acknowledgement
